leetcode日刷一题:
1.按类别刷，由易到难
2.如果题目难，可以一天分享思路，一天分享代码，或者要求更长时间
3.可以刷别人刷过的题，分享补充新的思路
4.如果有事太忙，可以请假不刷题，如果无故不刷题，就手推SVM或刷一道中等难度的题
总结与整理:
1.每周周一前，大家上传自己本周的代码和自己本周所有代码的思路总结，到自己的分支，我会合并分支到master，并且汇总大家的思路
2.代码命名格式:为了能快速找到可用的代码，leetcode题，用L打头，L题号中文题名 例如L70爬楼梯.cpp，笔试题用B打头，B笔试日期_公司_题名 例如 B8.12_头条_区间合并.cpp
如果是面试算法题，用M打头，M面试日期_公司_题名 例如 M8.20_头条_区间合并.cpp，如果是其它途径的题，直接写中文题名.cpp 如区间合并.cpp
3.思路命名格式:自己每周所有题的思路都总结在一个word文档里，命名为:姓名首字母_上传的日期_思路.docx,例如 s_8.26_思路.docx 
使用题名做每段标题，并在题的旁边注上本题类型，方便我汇总分类，例如动态规划，字符串，并查集等
当然也可以采取一级标题，二级标题的形式
